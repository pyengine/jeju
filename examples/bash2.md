# Environment

Environment gives replacable item of code.
In code, ${KEYWORD} is replaced by Environment value of *KEYWORD*

~~~bash
echo "print pwd"
pwd
echo "Environment is changed..."
~~~

Keyword | Value
---- | ----
OS  | Ubuntu
USER | sunshout
Target | Every One
PWD | /tmp


# Installation
